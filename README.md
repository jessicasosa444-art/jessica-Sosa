# jessica-Sosa
portfolio project for Intro to Programming course with Code the Dream 26.2
# about the project 
This portfolio website introduces who i am, the sills i have learned, and what i have created throughout the course.

The project includes:

- A personal portfolio website
- A Skills section generated with JavaScript
- A Projects section that uses the GitHub API
- A Leave a Message form with add and remove functionality
- A Weather App built with the Open-Meteo API

# Technologies Used

- HTML
- CSS
- JavaScript
- Git
- GitHub
- Fetch API
- Open-Meteo API

# Features

# Portfolio Page
- Navigation menu
- About section
- Experience section
- Skills section
- GitHub Projects section
- Connect section with GitHub and LinkedIn links
- Leave a Message form

# Weather App
- Current weather in New York, NY
- Daily weather forecast
- Error handling for failed API requests
